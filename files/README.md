# Marco Turrini - Personal Website

Professional portfolio website for wind energy and fluid mechanics research.

## File Structure

```
.
├── index.html              # Dark theme version
├── testindex.html          # Light theme version (recommended)
├── Marco_Turrini_CV.pdf    # Downloadable CV
└── images/                 # All image assets
    ├── light-ocean-waves.jpg
    ├── ocean-waves.jpg
    ├── wave-overlay.png
    ├── vortex-overlay.png
    ├── ripples-overlay.png
    ├── streamlines-overlay.png
    ├── turbulence-overlay.png
    └── ... (other images)
```

## Themes Available

### Light Theme (testindex.html) - RECOMMENDED
- Professional, clean design
- Better for academic/corporate contexts
- Excellent readability
- Subtle hydrodynamic visualizations

### Dark Theme (index.html)
- Modern, tech-forward aesthetic
- Blue accent colors
- Atmospheric feel

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Interactive Elements**: Hover effects on cards and sections
- **Transparent Overlays**: Beautiful flow visualizations
- **Smooth Animations**: Subtle motion for engagement
- **Downloadable CV**: Professional PDF for recruiters
- **Real Publications**: Live links to your research

## Usage

1. Upload all files maintaining the folder structure
2. Open `testindex.html` in your browser
3. All image paths use relative references: `images/filename.jpg`

## Customization

To update content, edit the HTML files:
- **Projects**: Section with id `#projects`
- **Publications**: Section with id `#publications`
- **Skills**: Section with id `#skills`
- **Research Areas**: Section with id `#research`

## Browser Compatibility

- Chrome/Edge (recommended)
- Firefox
- Safari
- Mobile browsers
